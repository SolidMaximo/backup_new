# BackupTool v1.0.0 by ONE Robotics Company

## Install

1. Extract zip file
2. Do one of the following with the appropriate version (32 or 64 bit):
   1. Copy to your current project's directory and rename to whatever you like (e.g. backup.exe)
   2. Extract somewhere and add that directory to your system's PATH variable

## USAGE

1. Create config file by running `BackupTool` once in your project directory
2. Use `BackupTool add` to add robots to your project
3. Use `BackupTool backup` to view a list of filters e.g. `BackupTool backup all` while grab everything off of all robots.

You can always simply type `BackupTool` for a reference on usage.
